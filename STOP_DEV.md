# Как запускать

Дважды кликните `start-dev.cmd` в корне проекта. Откроются отдельные окна для backend и frontend, а браузер сам перейдет на `http://localhost:5173`.

# Что должно открыться

1. Окно backend с `python -m uvicorn app.main:app --reload --port 8001`
2. Окно frontend с `npm run dev -- --host 127.0.0.1`
3. Браузер с `http://localhost:5173`

# Если backend не поднялся

В главном окне появится сообщение `Backend не запустился на 8001. Проверьте окно backend.`  
После этого посмотрите на окно backend: там обычно видно, что занялся порт или не поднялась база.

# Если порт 8001 занят

Значит, уже запущен другой backend. Закройте старый процесс на `8001` или освободите порт, затем запустите `start-dev.cmd` еще раз.

# Что не открывать

Не открывайте старый `http://127.0.0.1:8000`. Для этого проекта нужен `http://127.0.0.1:8001`.
